# Student Portal Management System

A complete ASP.NET Core MVC web application developed during internship training.

## Features

- Admin Login
- Student Management
- Attendance Management
- Attendance History
- Result Management
- Result History
- Fee Management
- Fee History
- Notice Board
- Dashboard Analytics
- Search Functionality
- Portal Data Reset

## Technologies Used

- ASP.NET Core MVC
- C#
- SQL Server
- Entity Framework Core
- Bootstrap 5
- HTML/CSS
- JavaScript
