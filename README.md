# Student Portal Management System

A complete ASP.NET Core MVC web application developed during internship training.

## Features

- Admin Login System
- Student Management
- Attendance Management
- Attendance History
- Result Management
- Result History
- Fee Management
- Fee History
- Notice Board
- Dashboard Analytics
- Search Functionality
- Reset Portal Data

## Technologies Used

- ASP.NET Core MVC
- C#
- Entity Framework Core
- SQL Server
- Bootstrap 5
- HTML
- CSS
- JavaScript

## Modules

### Student Module
- Add Student
- Edit Student
- Delete Student
- Upload Student Photo

### Attendance Module
- Mark Attendance
- Attendance History
- Attendance Analytics

### Result Module
- Result Entry
- Grade Calculation
- Result History

### Fee Module
- Fee Collection
- Due Fee Calculation
- Fee History

### Notice Module
- Add Notice
- Delete Notice
- Notice Board
